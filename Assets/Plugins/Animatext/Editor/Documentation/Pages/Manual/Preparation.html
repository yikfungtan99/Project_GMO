Preparation

Requirement

Animatext works with Unity 2019.4 or newer. If you need to use TextMeshPro version, please install the "TextMeshPro" package in the Package Manager.

Installation

Open your project. Go to "Project" panel, right-click and select "Import Package -> Custom Package", then select the Animatext package.


If you need to use TextMeshPro version, please import the package in the "Plugins -> Animatext -> Packages" folder.

  • TextMeshPro 1.5.0/2.1.0/3.0.0 or newerImport the "TextMeshPro" package.

  • TextMeshPro 1.5.0/2.1.0/3.0.0 olderImport the "TextMeshProOld" package.


Note: Please backup the project files before installation or update.
